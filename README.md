# FinTrack-Analysis

#  An Interactive Personal Finance Tracker for Budgeting, Savings, Investments, and Net Worth Analysis


**FinTrack-Analysis** is a full-stack personal finance management system designed to help users take control of their money.  
It combines **budgeting, expense tracking, savings, investments, and net worth calculation** into one powerful dashboard.  
The system also includes a **“Can I afford it?”** analysis feature that helps users make informed purchase decisions.

---

###  Features
-  **Budgeting & Expense Tracking** – Create monthly budgets and track spending by category.
-  **Savings Management** – Record and monitor your monthly savings progress.
-  **Investment Tracking** – Manage stock and cryptocurrency portfolios.
-  **Net Worth Calculation** – Calculate total assets minus liabilities in real-time.
-  **Data Visualization** – View financial summaries through dynamic charts.
-  **Secure Authentication** – JWT-based login system for data protection.
